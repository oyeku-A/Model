This is Breast Cancer Wisconsin (Diagnostic) Data Set (website: https://archive.ics.uci.edu/ml/machine-learning-databases/breast-cancer-wisconsin/).
The dependent variable is class (M = malignant represented by 4 in Dataset, B = benign represented by 2 in Dataset) 
7 supervised classification models using sklearn are included, with their confusion matrix accompanied with thier accuracy score.
Data2.csv was gotten directly from the site and then formatted into Data.csv to enable inputation into our various machine learning classification models.
